Place your scanned campfire log asset in this folder using one of these names:

- scanned-log.glb
- scanned-log.gltf
- scanned-log.obj
- scanned-log.fbx

Alternative subfolder names also supported:

- /models/log-scan/scanned-log.glb
- /models/log-scan/scanned-log.gltf
- /models/log-scan/scanned-log.obj
- /models/log-scan/scanned-log.fbx

At runtime, Firesim will auto-load the first available file from that list.
If none are found, it falls back to procedural cylinder logs.
